# 🧰 Toolkits Tampermonkey – CELK / SISREG3

Conjunto de **UserScripts (Tampermonkey)** para agilizar tarefas de **regulação e recepção**, integrando **SISREG3 → CELK → WhatsApp Web**.

## 🧩 Instalação

1. Instale a extensão **Tampermonkey** no Google Chrome:  
   👉 https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
2. Clique no ícone do Tampermonkey.
3. Vá em **Utilitário**.
4. Selecione **Adicionar novo script**.
5. **Importe todos os arquivos `.js`** do repositório.
6. Salve os scripts.

Após a instalação, os toolkits serão carregados automaticamente ao acessar os sistemas correspondentes.
